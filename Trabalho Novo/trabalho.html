Trabalho de Pesquisa: Introdução ao JavaScript

Questão 01: O que é o JavaScript? Onde ele pode ser utilizado em aplicações web? Explique a função do HTML, do CSS e do JavaScript em uma página web.

JavaScript é uma linguagem de programação dinâmica que permite criar interatividade nas páginas web. Ele é usado para validar formulários, modificar conteúdo e estilo de páginas, realizar requisições assíncronas, entre outras funcionalidades.

HTML (Hypertext Markup Language) define a estrutura e o conteúdo da página web (como texto, links, imagens).

CSS (Cascading Style Sheets) é responsável pela apresentação visual da página, como cores, fontes e layout.

JavaScript adiciona interatividade e dinâmica, permitindo a modificação da estrutura (HTML) e da aparência (CSS) de uma página de forma programática.

Questão 02: O que são APIs? Cite pelo menos três exemplos de APIs JavaScript que são bastante utilizadas, descrevendo o que elas permitem realizar.

APIs (Application Programming Interfaces) são interfaces que permitem a comunicação entre diferentes sistemas, oferecendo funcionalidades reutilizáveis.

  • API de Geolocalização: Permite obter a localização geográfica do usuário.
  • API Fetch: Permite fazer requisições HTTP assíncronas para buscar dados de servidores.
  • API do DOM: Permite a manipulação e interação com a estrutura HTML de uma página.

Questão 03: O que é o Document Object Model (DOM) de uma página web? O que é uma árvore DOM?

O DOM é uma interface que representa a estrutura de uma página HTML como uma árvore de nós, onde cada nó é um elemento HTML.

A árvore DOM é essa estrutura hierárquica que permite o acesso e manipulação de elementos da página via JavaScript.

Exemplo de métodos para acessar o DOM:

  • getElementById(): Retorna o elemento com o ID especificado.
  • getElementsByTagName(): Retorna todos os elementos de uma tag específica.
  • getElementsByClassName(): Retorna todos os elementos de uma classe específica.
  • querySelector(): Retorna o primeiro elemento que corresponde ao seletor CSS fornecido.
  • querySelectorAll(): Retorna todos os elementos que correspondem ao seletor CSS.

Questão 04: Explique a diferença das propriedades innerHTML, innerText e textContent do Document Object Model.

innerHTML: Retorna ou define o conteúdo HTML de um elemento, incluindo tags HTML.

innerText: Retorna ou define o texto de um elemento, sem incluir tags HTML.

textContent: Retorna ou define o conteúdo de texto de um elemento, sem considerar as tags HTML e respeitando a visibilidade do conteúdo.

Questão 05: De quais maneiras podemos inserir código JavaScript às páginas web? Explique cada uma delas.

Existem três formas principais de incluir JavaScript em uma página web:

  • Em linha (inline): O código JavaScript é escrito diretamente no atributo `onclick`, `onload`, etc., de um elemento HTML.
  • Interno: O código JavaScript é incluído dentro de uma tag <"Script"> diretamente no código HTML.
  • Externo: O código JavaScript é escrito em um arquivo separado (geralmente com extensão `.js`) e é vinculado ao HTML através da tag <"script src="arquivo.js">.

Questão 06: O que são eventos HTML? Cite pelo menos 5 exemplos de evento HTML e explique quando eles são disparados. Como um evento HTML pode ser utilizado pelo JavaScript para realizar uma determinada tarefa?

Eventos HTML são ações ou ocorrências que podem ser capturadas e tratadas pelo JavaScript para adicionar interatividade nas páginas web.

  • click: Disparado quando um elemento é clicado.
  • mouseover: Disparado quando o mouse passa sobre um elemento.
  • keydown: Disparado quando uma tecla é pressionada.
  • submit: Disparado quando um formulário é enviado.
  • load: Disparado quando uma página ou imagem é carregada.

Os eventos podem ser utilizados no JavaScript para acionar funções que alteram o comportamento da página, como modificar o conteúdo ou a aparência de um elemento.

Questão 07: Pode-se afirmar que a linguagem JavaScript é orientada a eventos e de tipagem dinâmica. Explique o que esses conceitos querem dizer e apresente as formas em que uma variável pode ser criada em JavaScript.

Orientada a eventos significa que o JavaScript reage a eventos como cliques ou carregamentos de página, realizando ações quando esses eventos acontecem.

Tipagem dinâmica significa que o JavaScript não exige a declaração do tipo de variável; o tipo é atribuído automaticamente com base no valor atribuído.

  • var: Cria uma variável que pode ter seu valor alterado ao longo do código.
  • let: Cria uma variável com escopo de bloco, ideal para controle de fluxo.
  • const: Cria uma variável constante, cujo valor não pode ser alterado depois de definido.

Questão 08: No contexto de escopo de variáveis, explique os conceitos de escopo global e local. Como criar variáveis de escopo global e local em JavaScript?

Escopo global: Variáveis definidas fora de funções ou blocos e que podem ser acessadas de qualquer parte do código.

Escopo local: Variáveis definidas dentro de uma função ou bloco e só podem ser acessadas dentro de sua função ou bloco.

  • Global: Variáveis definidas fora de funções.
  • Local: Variáveis definidas dentro de funções com `let`, `var` ou `const`.

Questão 09: Como podemos criar funções JavaScript em nossas páginas web? O que é uma função autoexecutável em JavaScript? Como podemos criá-la e quando ela é executada?

Funções em JavaScript podem ser criadas utilizando a palavra-chave `function` ou expressões de funções.

Função autoexecutável (IIFE - Immediately Invoked Function Expression) é uma função que se executa imediatamente após ser declarada.

                    (function() {
                        console.log('Executando!');
                    })();

Essa função é executada assim que é declarada e pode ser usada para criar um escopo isolado.

Questão 10: Como é possível estilizar elementos de uma páginas web de forma dinâmica utilizando JavaScript? Cite ao menos 3 exemplos de propriedades que podem ser alteradas pelo JavaScript.

O JavaScript pode alterar o estilo de elementos HTML manipulando as propriedades de estilo diretamente ou adicionando/removendo classes CSS.

  • style.backgroundColor: Altera a cor de fundo de um elemento.
  • style.color: Altera a cor do texto de um elemento.
  • style.fontSize: Altera o tamanho da fonte de um elemento.

Questão 11: Explique o que são loops em JavaScript e qual a diferença entre o loop for e o loop while.

Loops são estruturas de controle de fluxo que permitem executar um bloco de código repetidamente.

  • for: Usado quando sabemos o número de repetições. Exemplo: `for (let i = 0; i < 10; i++) {}`
  • while: Usado quando não sabemos o número de repetições, mas temos uma condição para continuar. Exemplo: `while (x < 10) {}`
